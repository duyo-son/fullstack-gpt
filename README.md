## requirements
* python 3.11.4
* pyenv

## start

### if you dont have /env .
```sh
python -m venv ./env
```

### activate envirements .
```sh
source env/bin/activate
```

### deactivate envirements .
```sh
deactivate 
```

### package install
```sh
pip install -r requirements.txt 
```

### comfirm .env
```sh
cp .env.sample .env
# OPENAI_API_KEY=<you shoud open api key>
```
